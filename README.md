# Dotfiles management system

## Installation
``bash
./install.sh
``

## Usage
``bash
#Configure setups repo path 
dotfilesctl config repo=/path/to/repo
#List setups
dotfilesctl list-setups
#Deploy a specific setup
dotfilesctl deploy <setup>
#List deployments
dotfilesctl list-deployments
``

## Help
``bash
dotfilesctl help
``
